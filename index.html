About Me

My name is James and I love building websites and web applications that combine fresh and clean aesthetics with fluid and intuitive user interfaces. I have multiple years of experience in coding with HTML, CSS and Javascript as well as more advanced frameworks including (but not limited to) Sass/SCSS, Node, Express and React. This experience include attending an intensive 6 month fullstack web development training program at UCLA extension and working as a web development intern at Wilshire Financial Network. I am also constantly trying to learn more and add to my skill set as a developer.
